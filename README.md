<h1 align="left"> JavaScript Practice Exercises</h1>

Welcome to my **JavaScript Practice** repository! This space is dedicated to sharpening my JavaScript skills through a variety of fundamental exercises, ranging from basic syntax and logic to array manipulation and string handling.

## About

This project contains a growing collection of exercises I've solved to reinforce my understanding of JavaScript. Each snippet is written with clarity and simplicity to help in learning and revisiting core concepts.

The exercises include:

- Variable declaration & operations  
- Conditional logic & loops  
- Math & number operations  
- Array & string manipulation  
- Functions & algorithmic thinking  
- Practical logic problems  

<p> I’m constantly adding new exercises as I continue to improve. </p>

## Highlights

Here are just a few examples of what you’ll find in this repository:

- Check if a number is even or odd  
- Swap two variables without a temp variable  
- Print multiplication tables  
- Sum of the first N natural numbers  
- Reverse a string or array  
- Palindrome checker  

## Technologies

This project is based entirely on:

- **JavaScript (ES6+)**
- **Node.js** (for running scripts locally via terminal)

> Run each file using:  
> `node basic.js`  
> `node advanced.js`
